# Job Portal App (MERN Stack)

A job portal application built with **MERN (MongoDB, Express, React, Node.js)** for browsing jobs, applying, and managing applications.

## Features

* User Authentication with JWT
* Browse and manage job listings
* Responsive design for all devices
* Employers can view/manage applications

## Technologies

* **Frontend:** React.js, Bootstrap
* **Backend:** Node.js, Express.js, MongoDB
* **Auth:** JWT, Bcrypt
* **Images:** Cloudinary
* **Deployment:** Vercel (frontend), Render (backend), MongoDB Atlas

## Setup

1. Clone repo:

```sh
git clone https://github.com/KashishDaharwal/Job_Chaiye.git
```

2. Install dependencies:

```sh
cd backend && npm install
cd ../frontend && npm install
```

3. Add environment variables in `/backend/config/config.env` (optional)
4. Run backend:

```sh
node server.js
```

5. Run frontend:

```sh
npm run dev
```

6. Visit [http://localhost:5173](http://localhost:5173)
